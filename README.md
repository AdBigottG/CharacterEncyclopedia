This project is a character encyclopedia built in C#. The program allows the user to select different options from a menu, and based on the selected option, the application displays different characters along with their descriptions.

The program uses conditional logic and user input to determine which character information to show. Each character entry contains a name and a short description, creating a small interactive database of characters. The project demonstrates basic programming concepts such as control structures, data organization, and user interaction in C#.
